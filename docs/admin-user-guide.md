# Admin User Guide (מדריך מנהל)

## כניסה

1. גש לכתובת `/admin/login`
2. הזן שם משתמש וסיסמה (מוגדרים במשתני סביבה)

## ניהול תוכן

- **זמני תפילות** - הוספה, עריכה ומחיקה
- **שיעורי תורה** - ניהול שיעורים שבועיים
- **אירועים** - אירועים עם תאריך, שעה ותמונה
- **גלריה** - אלבומים ותמונות
- **מקווה** - מידע, תקנון, מחירים ושעות
- **באנרים** - הודעות בראש האתר
- **הגדרות** - תוכן דף הבית
- **הודעות** - הודעות מטופס יצירת קשר
- **יומן פעולות** - מעקב אחר שינויים

## התנתקות

לחץ "התנתק" בתפריט הצד.
